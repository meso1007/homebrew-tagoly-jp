# Homebrew Tap for Tagoly-JP

このリポジトリは Tagoly-JP の Homebrew 配布用 Tap です。

## インストール

Homebrew を使って Tagoly-JP をインストールできます。

```bash
brew tap meso1007/tagoly-jp
brew install meso1007/tagoly-jp/tagoly-jp
```

## 動作確認
```bash
tagoly-jp --version
tagoly-jp
```

## 注意点
- macOS / Linux に対応
- Tap は公開リポジトリである必要があります
- バイナリは GitHub リリースにアップロード済みである必要があります
